## ⭐ Most Important Concepts

### 1. NAV (Net Asset Value)
```
NAV = Total Assets - Total Liabilities
NAV Per Unit = Total NAV ÷ Number of Units
```
**This appears in multiple forms on EVERY exam!**

### 2. PRS Account Structure
- **70% = Sub-Account A** (Locked until retirement at 55)
- **30% = Sub-Account B** (Flexible, once per year, 8% tax penalty)

### 3. Code of Ethics - 4 Core Principles
1. Honesty & Integrity
2. Professionalism
3. Avoidance of Conflict of Interest
4. Disclosure of Information

### 4. Financial Planning - 6 Steps
1. Establish relationship
2. Gather data
3. Analyze status
4. Develop recommendations
5. Implement
6. Monitor & review
